Detailed Description

Represents an audio buffer in memory.

It is not possible to play a sound with an AudioBuffer object alone. It is necessary to create an AudioSource object and assign a reference to an AudioBuffer to it. Many AudioSource objects can share the same AudioBuffer objects, allowing to save resources. An AudioBuffer object loads the whole audio data into memory. For large audio data this might be expensive. In that case, it is suggested to use audio streams and a StreamingAudioSource.
