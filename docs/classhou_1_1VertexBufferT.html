Detailed Description

template<typename T, bool dynamicStorage = false>
class hou::VertexBufferT< T, dynamicStorage >

Represents a concrete instance of a type of graphical memory buffer.

Template Parameters
Tthe type of objects stored in the buffer.
dynamicStorageif true, the buffer can be modified after creation. If false, this is not possible.
