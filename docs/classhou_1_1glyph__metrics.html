Detailed Description

Class describing the metrics of a glyph.

all metrics are internal expressed in 26.6 pixel format, which means the unit is 1/64th of pixel.

Parameters
sizethe size of the glyph bounding box.
horizontal_bearingthe horizontal bearing of the glyph, that is the position of the baseline of the glyph when rendering text horizontally.
horizontal_advancethe horizontal advance of the glyph, that is the horizontal offset to the next glyph when rendering text horizontally.
vertical_bearingthe vertical bearing of the glyph, that is the position of the baseline of the glyph when rendering text vertically.
vertical_advancethe vertical advance of the glyph, that is the vertical offset to the next glyph when rendering text vertically.
