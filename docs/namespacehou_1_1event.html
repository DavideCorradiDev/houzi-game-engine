HOU_SYS_API void hou::event::set_quit_callback ( quit_callback  f)

Sets the callback for quit events.

Quit events are generated when the last window is closed, or by external events.

By default, when the sys module is initialized, signal handlers that generate a quit event are registered for SIGINT and SIGTERM.
