Detailed Description

Allows reading metadata and glyphs from font raw data.

Supported formats: TTF, TTC, CFF, WOFF, OTF, OTC, PFA, PFB, CID-keyed Type 1, SFNT-based bitmap fonts, X11 PCF, Windows FNT, BDF, PFR.

all metrics are expressed in 26.6 pixel format, which means the unit is 1/64th of pixel.
