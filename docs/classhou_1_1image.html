template<size_t Dim, pixel_format PF>
template<size_t OtherDim, pixel_format OtherPF>
hou::image< Dim, PF >::image ( const image< OtherDim, OtherPF > &  other)
inline

Conversion constructor.

Creates an image from an image with a different number of dimensions and/or format. If the pixel format is different, each pixel is converted according to the conversion rule of the pixel class. If the number of dimensions of other is lower, the generated image has size 1 on all additional dimensions. It is not possible to create an image from an image with more dimensions.
