Detailed Description

template<typename T>
class hou::span< T >

Lightweight proxy for any contiguous container which can be represented by a pointer and a size.

a span does not contain any elements, it just references elements placed elsewhere in memory. The user must ensure that a span always points at a valid memory block.

Note that a span has different semantics than containers, as they don't hold any elements but just a reference to a memory address.

Template Parameters
Tthe value type. For constant containers it should be const value_type.
