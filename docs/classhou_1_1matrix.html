Detailed Description

template<typename T, size_t Rows, size_t Cols>
class hou::matrix< T, Rows, Cols >

Represents a matrix.

The size of the matrix is static and the object is allocated on the stack. The elements of a row are contiguous in memory and columns are contiguous in memory.

Template Parameters
Tthe scalar type.
Rowsthe number of rows.
Colsthe number of columns.
