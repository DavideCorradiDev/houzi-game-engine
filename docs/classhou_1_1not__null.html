Detailed Description

template<typename T>
class hou::not_null< T >

Wrapper for pointers which cannot be null.

not_null works transparently as a pointer, but the wrapped pointer cannot be assigned the nullptr value.

Template Parameters
Tthe pointer type. It can be a naked pointer or a smart pointer.
