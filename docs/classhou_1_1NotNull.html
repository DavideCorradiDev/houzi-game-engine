Detailed Description

template<typename PtrType>
class hou::NotNull< PtrType >

Wrapper for pointers which cannot be null.

NotNull works transparently as a pointer, but the wrapped pointer cannot be assigned the nullptr value.

Template Parameters
PtrTypethe pointer type. It can be a naked pointer or a smart pointer.
