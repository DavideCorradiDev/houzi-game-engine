Detailed Description

template<typename T>
struct hou::is_contiguous_container< T >

Template checker to determine if the type is a contiguous container.

The check is true if T has a member type named pointer and a member function named data, and if the type returned by data is pointer. This is in accordance to what happens with contiguous containers in the standard library.

Tthe type to be checked.
