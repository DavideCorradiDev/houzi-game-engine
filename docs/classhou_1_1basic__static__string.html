Detailed Description

template<typename CharT, size_t N, typename Traits = std::char_traits<CharT>>
class hou::basic_static_string< CharT, N, Traits >

Stack allocated string.

Template Parameters
CharTthe character type.
Nthe maximum number of characters allowed in the string. The string will allocate memory for N + 1 elements (the extra element is for the string terminating character '\0').
Traitsthe character traits.

template<typename CharT, size_t N, typename Traits >
hou::basic_static_string< CharT, N, Traits >::basic_static_string ( const CharT *  s)
noexcept

Constructs the string with a copy of the null-terminated character string pointed to by s.

If the length of s is greater than N, only the first N characters of s will be copied.

The behaviour is undefined if s does not point at an array of at least Traits::length(s) + 1 elements of CharT, including the case when s is a null pointer.

Parameters
sthe input null-terminated string.
