Detailed Description

Represents an audio buffer in memory.

It is not possible to play a sound with an audio_buffer object alone. It is necessary to create an audio_source object and assign a reference to an audio_buffer to it. Many audio_source objects can share the same audio_buffer objects, allowing to save resources. An audio_buffer object loads the whole audio data into memory. For large audio data this might be expensive. In that case, it is suggested to use audio streams and a streaming_audio_source.
