Detailed Description

template<typename T>
class hou::rotation3< T >

Represents a rotation in 2d space.

The rotation is internally represented as a unit quaternion.

Template Parameters
Tthe scalar type.

template<typename T >
constexpr hou::rotation3< T >::rotation3 ( const mat3x3< T > &  m)
explicit

Creates a rotation with the given rotation matrix.

If m is not a valid rotation matrix, the matrix will be converted to a quaternion which will then be normalized. This will result in a valid rotation, whose rotation matrix is a different one from the one provided. The caller must therefore ensure that m is a valid rotation matrix.

Parameters
mthe rotation matrix.
