Detailed Description

template<typename T>
class hou::MeshT< T >

Represents a mesh.

The vertex information of a Mesh object is stored in the VRAM. There must be a current context to create a Mesh object. A Mesh object should be used only if the owning context is current.

Template Parameters
Tthe type of vertex stored by the mesh.
