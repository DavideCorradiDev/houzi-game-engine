Detailed Description

Class describing the metrics of a glyph.

All metrics are internal expressed in 26.6 pixel format, which means the unit is 1/64th of pixel.

verticalBearingthe vertical bearing of the glyph, that is the position of the baseline of the glyph when rendering text vertically.
